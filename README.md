# Alpha Level Website

AI-Powered Threat Detection marketing site.

## Structure

```
alphalevel/
├── index.html      # Main HTML
├── style.css       # All styles
├── script.js       # Animated pipeline + interactions
└── fonts/          # Inter font files (self-hosted)
```

## Local Development

Just open `index.html` in a browser — no build step needed.

## Deployment

Connected to Netlify. Push to `main` branch to auto-deploy.
